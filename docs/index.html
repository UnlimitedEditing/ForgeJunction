Interface

BUILT FOR SPEED

Say goodbye to clunky web interfaces. Forge Junction brings the power of cloud generation into a snappy, native desktop experience.

⚡
Native Performance
Built on lightweight Electron, the interface responds instantly. No browser tabs to lose, no accidental refreshes during a massive batch run.
☁
Cloud Powered
Generation runs on Graydient's dedicated GPU clusters. Your computer stays quiet and cool while rendering high-res media.
🔐
Secure Credentials
Your API token is stored encrypted in your OS keychain — Windows Credential Manager or macOS Keychain. Never in plaintext.

Getting Started

INSTALLATION

Forge Junction connects to your Graydient account via an API token. Here's how to get set up in under 2 minutes.

1
Get your API Token
Log into Graydient.ai, navigate to your Account Settings, and generate a new API token. Keep this safe.
2
Download & Launch
Grab the recommended installer from the right, run it, and open Forge Junction.
3
Authenticate
On first launch, the app will ask for your token. Paste it in, and you're ready to start rendering.
⬇ Installer (Windows)
Recommended
Download the latest .exe from the Releases page, run it, and launch. No Node.js required.
⚙ Run from Source
Requires Node.js 18+. Clone the repo and double-click launch.bat — it handles everything automatically.
# Windows — just double-click launch.bat
git clone https://github.com/UnlimitedEditing/ForgeJunction
cd ForgeJunction
# Or manually:
npm install
npm run dev
🔧 Production Build
Package the app yourself. Outputs an NSIS .exe installer to release/.
npm run build
npm run package
