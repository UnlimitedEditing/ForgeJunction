WHAT'S NEW

A massive leap forward in how you interact with the Graydient API — from simple prompt-and-render to complex, multi-stage generation pipelines.

🖥
Unified UI
The entire interface has been redesigned. Most functionality you know from Telegram PD is now fully integrated and accessible from the main application window.
UI Overhaul
⬡
Node Editor
A brand-new visual editor for planning and executing complex workflows. The output of one node seamlessly becomes the input for the next, with quick form templates for repeatable setups.
New Feature
✂

A built-in lightweight NLE to trim and assemble your video generations without leaving the app. Cut, arrange, and preview clips right inside Forge Junction.
New Feature

Asset Browser
Dedicated Concept and LoRA browsing directly in the UI. Find styles, fine-tunes, and concepts interactively and wire them straight into your node graph.

Interface

BUILT FOR POWER

Say goodbye to clunky web interfaces. Forge Junction brings the full power of cloud generation into a snappy, native desktop experience — now with multi-stage pipelines and a built-in editor.

⬡
Node-Based Pipelines
Chain generations together visually. Route the output of a Flux render straight into an upscaler node, a ControlNet, or a video model — no manual file juggling.
☁
Cloud Powered
Generation runs on Graydient's dedicated GPU clusters. Your computer stays quiet and cool while rendering complex, multi-node pipelines.
✂
Built-in Video NLE
Trim, arrange, and preview your AI video generations right inside the app. No need to open a separate editor for basic assembly work.
🗂
Live Asset Browsing
Browse, search, and preview thousands of LoRAs and concepts from inside the UI. Click to wire any asset directly into your current node graph.
⚡
Native Performance
Built on lightweight Electron, the interface responds instantly. No browser tabs to lose, no accidental refreshes killing a long batch run.
🔐
Secure Credentials
Your API token is stored encrypted in your OS keychain — Windows Credential Manager or macOS Keychain. Never in plaintext.

Getting Started

INSTALLATION

Forge Junction connects to your Graydient account via an API token. Get set up in under 2 minutes.

1
Get your API Token
Log into Graydient.ai, navigate to Account Settings, and generate a new API token. Keep this safe.
2
Download & Launch
Grab the recommended installer from the right, run it, and open Forge Junction. No dependencies needed.
3
Authenticate
On first launch, paste your API token when prompted. Future updates arrive silently in the background via the built-in auto-updater.
⬇ Installer (Windows)
Recommended
Download the latest .exe from the Releases page, run it, and launch. No Node.js required.
⚙ Run from Source
Requires Node.js 18+. Clone the repo and double-click launch.bat — it handles everything automatically.
# Windows — just double-click launch.bat
git clone https://github.com/UnlimitedEditing/ForgeJunction
cd ForgeJunction
# Or manually:
npm install
npm run dev
🔧 Production Build
Package the app yourself. Outputs an NSIS .exe installer to release/.
npm run build
npm run package
🍎 Mac Support
Need help running Forge Junction on macOS? Contact @UnlimitedEditing on Telegram for dedicated support.
🐛 Found a Bug?
Use Help → Export Debug Report, attach the file, and open a GitHub issue. The report includes system info, app version, and relevant log output — it makes fixes much faster.
