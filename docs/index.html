Changelog

WHAT'S NEW

A big one. InfiniteCanvas lands, Chain Builder grows up, and Skills becomes the default way everything works. Also a lot of UI polish — probably too much to list honestly.

∞
Infinite Canvas
A brand-new free-form workspace. Spawn nodes anywhere, type prompts, get media. Use it to notarise ideas, moodboard compositions, or ideate without committing to structure. This is the foundation for rooms and collaboration.
New Feature
⬡
Chain Builder → Custom Nodes
Chain Builder now creates real custom nodes for the InfiniteCanvas. Chain prompts, pick variable fields, spawn the whole thing as a single node. Collaborators see only the result — your pipeline stays yours.
Major Upgrade
✦
Skills as Default Input
Open the app and start typing — Skills is now the default. No workflow selection, no config. Same for the node graph. Pick an explicit method if you want to, or just let it run. It's magic, basically.
New Feature
🎨
UI & Animations Overhaul
Far too many UI optimisations to list with a straight face. The interface is faster, tighter, and more considered. Animations added throughout, plus a proper startup loading screen — ForgeJunction looks like it means business from second one.
Polish
🗂
Media Library Fixed
The library now correctly surfaces all your renders — no more missing outputs. It should have always worked this way. It does now.
Bug Fix

New Feature

BUILT FOR POWER

Open the app, start typing. Skills gets you there. When you need more, the node graph, the Chain Builder, and the InfiniteCanvas are waiting — one app, every tool, cloud-powered rendering your machine never has to think about.

∞
Infinite Canvas
A free-form working space for your creative process. Spawn nodes anywhere, type prompts, get media back. Moodboard, ideate, notarise — without worrying about structure. Room hosting and multiplayer are coming here next.
⬡
Custom Node Builder
Chain Builder now builds real custom nodes. Compose multi-prompt pipelines, expose specific fields as variable inputs, and collapse the whole thing down to a single spawnable node on the InfiniteCanvas.
✦
Skills — Just Start Typing
Skills is now the default input everywhere — the main prompt bar and the node graph. Fire up the app, type what you want, and it handles the rest. You don't have to pick a workflow to get started.
☁
Cloud Powered
Generation runs on Graydient's dedicated GPU clusters. Your machine stays quiet while rendering complex, multi-node pipelines. The heavy work lives in the cloud — the experience lives on your desktop.
🗂
Media Library — All Renders
The library now finds everything. Browse all your renders with proper NSFW flagging and a global hide toggle in settings. Your full output history, finally reliable.
🔐
Native & Secure
Built on Electron. Responds instantly, no browser tabs to lose, no refreshes killing a long batch. Your API token is encrypted in your OS keychain — Windows Credential Manager or macOS Keychain. Never plaintext.

Getting Started

INSTALLATION

Forge Junction connects to your Graydient account via an API token. Get set up in under 2 minutes.

1
Get your API Token
Log into Graydient.ai, navigate to Account Settings, and generate a new API token. Keep this safe.
2
Download & Launch
Grab the recommended installer from the right, run it, and open Forge Junction. No dependencies needed.
3
Authenticate
On first launch, paste your API token when prompted. Future updates arrive silently in the background via the built-in auto-updater.
⬇ Installer (Windows)
Recommended
Download the latest .exe from the Releases page, run it, and launch. No Node.js required.
⚙ Run from Source
Requires Node.js 18+. Clone the repo and double-click launch.bat — it handles everything automatically.
# Windows — just double-click launch.bat
git clone https://github.com/UnlimitedEditing/ForgeJunction
cd ForgeJunction
# Or manually:
npm install
npm run dev
🔧 Production Build
Package the app yourself. Outputs an NSIS .exe installer to release/.
npm run build
npm run package
🍎 Mac Support
Need help running Forge Junction on macOS? Contact @UnlimitedEditing on Telegram for dedicated support.
🐛 Found a Bug?
Use Help → Export Debug Report, attach the file, and open a GitHub issue. The report includes system info, app version, and relevant log output — it makes fixes much faster.
